HOW TO INSTALL UBUNTU THROUGH USB?

Hi, Sumedh here...Today i am going to show you how to install ubuntu on windows running pc through bootable ubuntu pendrive!

  1. Firstly insert a 4gb (or more) storage pendrive to ur laptop.Make sure u have removed all of its data and put somewhere because the PENDRIVE WILL BE FORMATTED!
  2. Next install latest ubuntu pendrive bootable OS from the official website ONLY!
  3. Now at the same time install YUMI software fromYUMI website...
  4. After the installation of all the required things, run the YUMI software and follow the below procedures:
    • Select the pendrive to which u have to put the OS..
    • Select FreeDos as boot and select the iso file by clicking SELECT
    • Now dont go to change anything else[Next 2 options down there are by default]
    • Click on start button
    • Click Yes to all dialogue boxes and select the recommended options!{Do not select the DD convertion but Select the ISO **conversion option**}
    • It will take about 5-10 minutes to complete the setup...If u want to put another OS to ur pendrive, u can click yes when YUMI asks u else no..I am selecting "no"
    • After the installation is complete, click finish and eject the pendrive
    • Now u have the bootable Ubuntu on ur Pendrive! :)

    HOW TO USE IT NOW???

    1. On ur computer/laptop/desktop..whatever...and insert ur pendrive before its startup...
    2. Now press the required key to enter Multiboot Menu..[it is f12 in my case]
    3. Once entered select ur pendrive
    4. Then select the name that starts with "Linux..."
    5. Now select ubuntu from the list
    6. Then select first option[i.e. the name that DOESN'T have any extras like the shortest name among the 2]
    7. Then it starts a command line type and then scans ur drive for errors[better to allow the process to complete]
    8. Now ubuntu desktop window opens and you are ready to use UBUNTU
